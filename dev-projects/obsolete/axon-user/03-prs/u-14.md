# U-14 — `tools/docgen_verify.py` enforces tier link discipline

**Wave**: U.E · **Depends-on**: U-10, U-11, U-12, U-13 · **Status**: ready

## Why

Without enforcement, the hierarchy decays. Authors will skip the
`Parent-phase:` line on PRs; phase docs will forget their roadmap pointer;
ADRs will drift unlinked. A single linter rule prevents this.

## Evidence

- U-10..U-13 introduce three new doc kinds (roadmap, phase, ADR) and a new
  PR field (`Parent-phase:`). All four need a verifier.
- `tools/docgen_verify.py` already exists and runs in `_check-all.sh` —
  extending it costs no new file.

## Design notes

- New checks (added to existing `docgen_verify.py`):
  1. If `02-phases/` exists in a project: every `03-prs/pr-*.md` MUST have
     `Parent-phase:` pointing to an existing phase file.
  2. Every `02-phases/phase-*.md` MUST link to its parent roadmap
     (`02-roadmap.md`) IF the roadmap exists.
  3. Every `03-decisions/adr-*.md` SHOULD link to ≥ 1 PR or phase
     (warning, not error — ADRs can be standalone).
- Only enforces when the relevant directory exists (legacy flat plans are
  skipped — see v3 plan §7 risk mitigation).

## Pitfalls

- **R (false positives on legacy)**: scoping the checks to "only when dir
  exists" handles this.
- **F (slow verifier)**: each check is a directory glob + file head read —
  negligible.

## Interface sketch

```bash
$ python3 tools/docgen_verify.py
{
  "ok": true,
  "checks": {
    "schema_links":     "pass (8 docs)",
    "phase_link":       "pass (4 PRs reference parent phase)",
    "roadmap_link":     "pass (3 phases reference roadmap)",
    "adr_link":         "warn (1 ADR without PR/phase link: adr-003)"
  }
}
```

## Spec

### Files

- `tools/docgen_verify.py` — add `check_phase_link`, `check_roadmap_link`,
  `check_adr_link` functions; wire into main.

### Acceptance

- Running against a fully-linked v3 project exits 0.
- Running against a project where a PR is missing `Parent-phase:` exits non-zero
  with a clear error message naming the file.
- Running against a legacy axon-master project (no `02-phases/` dir) exits 0
  (skipped).

### Rollback

`git revert`. Existing schema-link check unaffected.
