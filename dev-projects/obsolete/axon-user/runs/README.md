# axon-user — runs index

Each `runs/R-{NNN}-{persona}-{workflow}.md` captures one in-character
persona run: the transcript, the observed behavior, the findings opened.

The transcript is written by the **driver agent** acting as the persona
— first-person, in-character voice, no agent narration.

## Format

```
# R-001 · P1-novice-naomi × W-01

## context
- starting state: cold boot
- patience: 6 turns

## transcript
> [persona-voice prompt]
< [AXON output]
> ...

## summary
- result: pass | fail | abandoned at turn N
- findings: F-001 (S2 — identity-gate wording)
```
