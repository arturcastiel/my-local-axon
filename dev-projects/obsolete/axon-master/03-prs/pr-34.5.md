# pr-34.5 — Cheatsheet auto-section via docgen

**Wave**: W4 · **Goals**: G.umb.06 · **Depends-on**: PR-23 (docgen infra), PR-13 (usage frequency)

## Why
Cheatsheet (PR-6) needs to stay current as verbs come and go. Auto-generation from program `# desc:` lines + usage-log frequency keeps it accurate without human curation.

## Evidence
- `helpers/cd-gap-c1-p3-goals-extracted.md` → G.umb.06 "cheatsheet auto-section".
- `helpers/cd-wf-c4-p2-roadmap.md` Wave 4 PR-4.5.

## Design notes
- `tools/docgen.py` gains an `update-cheatsheet` subcommand.
- Reads every `# desc:` line from `workspace/programs/code-dev*.md` whose `status == active`.
- Sorts by usage-log frequency (top 10 by run count, last 30 days) or alphabetically within umbrella.
- Replaces content between markers in `workspace/AXON-DOCS-CHEATSHEET.md`:
  ```
  <!-- AUTO-VERBS-START -->
  …
  <!-- AUTO-VERBS-END -->
  ```
- Idempotent: same input → same output.
- Cheatsheet remains ≤ 80 lines.

## Pitfalls
- `# desc:` lines vary in quality → docgen warns when too long (> 60 chars).
- Markers missing → `update-cheatsheet` exits 1 with explanatory message.

## Interface sketch

```
$ python3 tools/docgen.py update-cheatsheet
OK: 14 verbs regenerated, byte-identical on rerun. Cheatsheet at 73/80 lines.
```

## Spec
- **Files**: modified `tools/docgen.py`, `workspace/AXON-DOCS-CHEATSHEET.md` (markers).
- **Acceptance**:
  1. `docgen update-cheatsheet` emits VERBS section between markers.
  2. Sorted by usage-frequency (top 10) or umbrella.
  3. Idempotent regeneration.
  4. Cheatsheet ≤ 80 lines after generation.
  5. `tools/lint_paths.py` clean.
- **Rollback**: revert (cheatsheet keeps last generated content; manual edits possible).
- **Owner**: AGENT writes; HUMAN runs once and commits.

## Codebase grounding
- **modify**: [`tools/docgen.py`](../../../../tools/docgen.py) — add `update-cheatsheet` subcommand:
  - Walks [`workspace/programs/code-dev*.md`](../../../../workspace/programs/), extracts `# desc:` line, status filter `status != deprecated`.
  - Reads [`tools/usage.py`](../../../../tools/usage.py) `top --kind program --window 30d --limit 10` (JSON).
  - Renders verbs section sorted by frequency (top 10) + remainder alphabetical-within-umbrella.
  - Replaces content between markers in [`workspace/AXON-DOCS-CHEATSHEET.md`](../../../../workspace/AXON-DOCS-CHEATSHEET.md) (`<!-- AUTO-VERBS-START -->` to `<!-- AUTO-VERBS-END -->`; markers seeded in PR-6).
- **idempotent**: rerun → same bytes when inputs unchanged. Tested by running twice and comparing.
- **size guard**: total cheatsheet ≤ 80 lines after generation.
- **WARN**: `# desc:` lines > 60 chars.
