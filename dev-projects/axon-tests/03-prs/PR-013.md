  `.flow.jsonl` fixture and asserts user-visible output + state

  per the catalog.

### tests/fixtures/workflows/w0[1-7].flow.jsonl  (7 new fixtures)
- **What:** Sequenced replays. Each `{input, expected_state,
  expected_files}` represents a single workflow step.
