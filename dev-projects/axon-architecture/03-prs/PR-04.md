# PR-04 · Hook wrappers (make the response gate installable) (Theme A) · AUTO wrappers / OWNER install

## Closes (the "wrappers don't exist" half of)
- **F02 / F09 / F52** — `.claude/settings.json.proposed` references 3 `tools/hooks/` wrappers that do not
  exist, so the per-turn enforcement keystone is un-installable as documented.

## Change
Author `tools/hooks/` (a hooks adapter dir — NOT registered tools; invoked by the host hook config, so
out of scope for R_NO_ORPHAN_TOOLS / REGISTRY):
- `reanchor_store.py` — UserPromptSubmit: stdin JSON → prompt → `memory.py set --scope W --key
  recent-user-input` (feeds anticipate). Exit 0.
- `enforce_pretooluse.py` — PreToolUse(Write/Edit/NotebookEdit/MultiEdit): stdin JSON → target path →
  `enforce.py check-write --target`; on block (exit≠0) → stderr + **exit 2** (deny the tool).
- `verify_stop.py` — Stop: stdin JSON → transcript_path → last assistant text → `verify.py output
  --text`; on BLOCK (exit≠0) → stderr + **exit 2** (surface the violation).
- `enable-enforcement.sh` — opt-in installer: checks the 3 wrappers exist, merges the `.proposed` hook
  block into the active `.claude/settings.json`, reports what it activated. **Owner runs this.**

## Design
Each wrapper factors a **pure** payload→decision function (`extract_prompt`, `target_path`,
`last_assistant_text`) from a thin `main()` (stdin/subprocess/exit) — the pure cores are unit-tested; the
host-payload field names are best-effort (documented owner-verify point, per HOOKS-README).

## Tests (tests/test_hooks.py)
- `target_path` returns the file for a Write/Edit payload, None for a read tool → PreToolUse allows reads.
- `extract_prompt` pulls prompt across known key variants.
- `last_assistant_text` returns the final assistant message from a transcript jsonl.
- exit-code contract documented; payload-schema verification is the owner's install step.

## Gate / handoff
Wrappers + tests are AUTO (gate green → merge). **Installation/activation + flag-flip is the OWNER step**
(batched into the final owner hand-off with PR-3K + PR-24).
