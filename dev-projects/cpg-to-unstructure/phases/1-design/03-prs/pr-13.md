## Detailed specs

### `README.md` (top-level skeleton)

```markdown
# cpg2unstructured

Lightweight Python library: read Eclipse CPG (corner-point geometry)
`.grdecl` files → unstructured mesh + cell-cell adjacency graph for
shortest-path algorithms. NumPy-only by default; optional scipy CSR
backend for fast Dijkstra.

**Status:** v0.1.0 — alpha, matching-Z grids only (faulted geometry deferred to v2).
**Distribution:** local editable install only — not on PyPI.

## Install

```bash
git clone <repo>
cd cpg2python
python3 -m venv .venv && . .venv/bin/activate
pip install -e .            # numpy-only
pip install -e .[fast]      # + scipy.sparse for fast Dijkstra
pip install -e .[dev]       # + pytest, pytest-cov, build
```

## 30-second quickstart

```python
from cpg2unstructured import (
    read_grdecl, from_grdecl, build_topology, attach_props,
    cell_adjacency, shortest_path,
)

g = read_grdecl("examples/quickstart.grdecl")
grid = attach_props(build_topology(from_grdecl(g)), g)

print(grid.cart_dims, grid.n_cells)        # (5, 5, 3)  ~70 active cells
print(grid["PORO"].mean())                  # ~0.18

adj = cell_adjacency(grid, weights="unit")
dist, path = shortest_path(adj, 0, grid.n_cells - 1)
print(f"shortest path: {len(path)} cells, distance {dist}")
```

## API map

| Function / class | Purpose | PR |
|---|---|---|
| `read_grdecl(path) → Grdecl` | parse .grdecl deck | PR-3, PR-4 |
| `from_grdecl(grdecl) → CornerCube` | corner-cube + node dedup | PR-5 |
| `build_topology(cube) → UnstructuredGrid` | full mesh | PR-7 |
| `attach_props(grid, grdecl) → UnstructuredGrid` | properties on active cells | PR-12 |
| `face_geometry(...)`, `cell_geometry(...)` | centroids, areas, volumes | PR-8 |
| `IndexMap(cart_dims, ijk)` | active-id ↔ (i,j,k) | PR-9 |
| `cell_adjacency(grid, weights=...)` | adjacency list (numpy) | PR-10 |
| `cell_adjacency_csgraph(grid, ...)` | adjacency CSR (scipy) | PR-11 |
| `shortest_path(adj, src, dst)` | Dijkstra (numpy) | PR-10 |
| `shortest_path_fast(csr, src, dst=None)` | Dijkstra (scipy) | PR-11 |

## Limitations (v0.1.0)

- Matching-Z grids only. Faulted (non-matching-Z) decks raise
  `NonMatchingZError`.
- No coordinate transforms / projections.
- No transmissibility / well models.
- ASCII GRDECL only (Eclipse spec).
- Single-process, in-memory; no streaming for huge decks.

### `examples/quickstart.py`

```python
"""End-to-end demo: parse → mesh → graph → shortest path.

Runs in well under 1 s on a normal laptop.
"""
from __future__ import annotations

from pathlib import Path

import numpy as np

from cpg2unstructured import (
    read_grdecl, from_grdecl, build_topology, attach_props,
    cell_adjacency, shortest_path,
    cell_geometry,
)

HERE = Path(__file__).parent


def main() -> None:
    g = read_grdecl(HERE / "quickstart.grdecl")
    print(f"loaded {g.cart_dims} grid, {g.n_cells} cells "
          f"({int(g.actnum.sum())} active)")

    cube = from_grdecl(g)
    grid = attach_props(build_topology(cube), g)
    print(f"unstructured: {grid.n_cells} active cells, "
          f"{grid.faces.nodes.shape[0]} faces")

    cg = cell_geometry(grid.nodes, grid.faces, grid.cell_faces)
    print(f"total volume: {cg.volumes.sum():.3f}")

    if "PORO" in grid:
        print(f"PORO  mean: {grid['PORO'].mean():.3f}, "
              f"min: {grid['PORO'].min():.3f}, "
              f"max: {grid['PORO'].max():.3f}")

    adj = cell_adjacency(grid, weights="distance",
                         cell_centroids=cg.centroids)
    dist, path = shortest_path(adj, 0, grid.n_cells - 1)
    print(f"shortest distance from cell 0 to cell {grid.n_cells - 1}: "
          f"{dist:.3f}  ({len(path)} cells visited)")


if __name__ == "__main__":
    main()
```

### `examples/quickstart.grdecl`

A 5×5×3 cartesian deck with mild PORO heterogeneity and 5 inactive
cells (to exercise filtering). Generated by hand or by a one-off
script committed alongside; the file is checked in directly so the
example is self-contained.
