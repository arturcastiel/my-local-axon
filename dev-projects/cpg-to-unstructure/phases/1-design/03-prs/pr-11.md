# PR-11 — Optional `[fast]` scipy CSR + Dijkstra

> Wave 5 · Theme: graph layer
> Branch: `pr-11-graph-fast`
> Status: draft (not yet implemented)
> Dependencies: PR-10 (merged)

## One-liner
Add a scipy-backed shortest-path path that's a drop-in replacement for
PR-10's pure-numpy version: `cell_adjacency_csgraph` returns a
`scipy.sparse.csr_array`; `shortest_path_fast` calls
`scipy.sparse.csgraph.dijkstra`. Identical numerical results to PR-10
on every shared test grid.

## Why this PR
The numpy-only Dijkstra in PR-10 is fine for ≤ 10k cells but starts to
hurt at reservoir scale. scipy's `csgraph.dijkstra` is C-level and 10–100×
faster. We expose it as an **optional** path under the existing `[fast]`
extra so the core install stays numpy-only.

The two paths must agree to floating-point tolerance — that's the whole
point of having both, and what the v0.1.0 README will promise.

## Decisions baked in

- **scipy is OPTIONAL.** Imports are gated by a try/except at module
  import time; functions raise `ImportError` with a helpful message if
  called without scipy installed. Tests that exercise the fast path are
  marked `pytest.importorskip("scipy")`.
- **CSR built from `face_cells` directly**, not by walking PR-10's
  list-of-list. Saves memory and an extra pass.
- **Same weight modes as PR-10**: `"unit"`, `"distance"`, `"none"`.
- **`shortest_path_fast(csr, src, dst=None)`**: when `dst is None`,
  returns the full `(distances, predecessors)` arrays from scipy; when
  `dst is int`, reconstructs and returns `(dist, path)` like PR-10.
- **Numerical agreement test runs on a 10×10×5 grid.** Cells 0 → last,
  unit + distance weights. Tolerance: distances `np.allclose` with
  `rtol=0, atol=1e-12`; paths must be identical for unit weights, may
  differ for distance weights when ties exist (test asserts on cost
  equality, not path equality, in that case).
- **No CSC, no LIL.** CSR only — that's what `csgraph.dijkstra`
  prefers internally.

## Detailed specs

### `cpg2unstructured/graph.py` (additions)

```python
# at top of module
try:
    import scipy.sparse as _sp
    import scipy.sparse.csgraph as _csgraph
    _HAS_SCIPY = True
except ImportError:
    _HAS_SCIPY = False


def _require_scipy() -> None:
    if not _HAS_SCIPY:
        raise ImportError(
            "scipy is required for the [fast] code path. "
            "Install with: pip install cpg2unstructured[fast]"
        )


def cell_adjacency_csgraph(
    mesh: "_TopoMesh",
    weights: WeightMode = "unit",
    cell_centroids: np.ndarray | None = None,
):
    """Build the cell adjacency as a scipy.sparse.csr_array.

    Returns a symmetric CSR with ``shape = (n_active, n_active)``.
    """
    _require_scipy()
    if weights == "distance" and cell_centroids is None:
        raise ValueError("weights='distance' requires cell_centroids")

    nb = mesh.faces.neighbours
    mask = (nb[:, 0] != -1) & (nb[:, 1] != -1)
    a = nb[mask, 0]
    b = nb[mask, 1]
    if weights == "distance":
        w = np.linalg.norm(cell_centroids[a] - cell_centroids[b], axis=1)
    else:
        w = np.ones(a.shape[0], dtype=np.float64)

    n = mesh.cell_faces.shape[0]
    rows = np.concatenate([a, b])
    cols = np.concatenate([b, a])
    data = np.concatenate([w, w])
    return _sp.csr_array((data, (rows, cols)), shape=(n, n))


def shortest_path_fast(csr, src: int, dst: int | None = None):
    """scipy-backed Dijkstra. Drop-in for PR-10's shortest_path.

    If ``dst`` is None: return ``(distances, predecessors)`` arrays
    (full-source mode). If ``dst`` is an int: return ``(dist, path)``
    matching PR-10's signature.
    """
    _require_scipy()
    distances, predecessors = _csgraph.dijkstra(
        csr, directed=False, indices=src,
        return_predecessors=True,
    )
    if dst is None:
        return distances, predecessors

    if not np.isfinite(distances[dst]):
        raise ValueError(f"no path from {src} to {dst}")
    path = [dst]
    while path[-1] != src:
        path.append(int(predecessors[path[-1]]))
    path.reverse()
    return float(distances[dst]), path
```

### `tests/test_graph_fast.py`

```python
import numpy as np
import pytest

scipy = pytest.importorskip("scipy")  # noqa: F841 — gates the entire module

from cpg2unstructured import (
    cell_adjacency, shortest_path,
)
from cpg2unstructured.graph import (
    cell_adjacency_csgraph, shortest_path_fast,
)


def _build_NxNxN_mesh(n: int):
    """Helper: synthesize an N×N×N CornerCube + topology mesh."""
    from cpg2unstructured._toy import build_toy_cpg
    from cpg2unstructured.cpg import from_grdecl
    from cpg2unstructured.unstructured import build_topology
    # build_toy_cpg is N×N×N for any (n, n, n); reuse for the test.
    cpg = build_toy_cpg(n, n, n)
    # Wrap it as a Grdecl-shaped fake — for the test we go through
    # the same path PR-7's tests use. (See implementation note.)
    raise NotImplementedError("see implementation notes — adapter helper TBD")


def test_distance_agree_10x10x5():
    mesh, centroids = _build_NxNxN_mesh(10)  # also returns 5-deep variant
    # Slow path
    adj = cell_adjacency(mesh, weights="distance", cell_centroids=centroids)
    d_slow, _ = shortest_path(adj, 0, mesh.cell_faces.shape[0] - 1)
    # Fast path
    csr = cell_adjacency_csgraph(mesh, weights="distance",
                                 cell_centroids=centroids)
    d_fast, _ = shortest_path_fast(csr, 0,
                                   mesh.cell_faces.shape[0] - 1)
    np.testing.assert_allclose(d_fast, d_slow, atol=1e-12)


def test_unit_weight_path_identical_2x2x2(mesh_2x2x2):
    adj = cell_adjacency(mesh_2x2x2, weights="unit")
    d_slow, p_slow = shortest_path(adj, 0, 7)
    csr = cell_adjacency_csgraph(mesh_2x2x2, weights="unit")
    d_fast, p_fast = shortest_path_fast(csr, 0, 7)
    assert d_fast == d_slow == 3.0
    # Path equality holds for unit weights with this start/end on 2x2x2
    assert p_slow == p_fast


def test_full_source_mode(mesh_2x2x2):
    csr = cell_adjacency_csgraph(mesh_2x2x2, weights="unit")
    distances, _ = shortest_path_fast(csr, 0, dst=None)
    assert distances.shape == (8,)
    assert distances[0] == 0.0
    assert distances[7] == 3.0


def test_unreachable_raises(mesh_2x2x2):
    csr = cell_adjacency_csgraph(mesh_2x2x2, weights="unit")
    # Zero out row 7 to break it off — scipy CSR isolated cell behaviour
    csr_lil = csr.tolil()
    csr_lil[7, :] = 0
    csr_lil[:, 7] = 0
    csr2 = csr_lil.tocsr()
    csr2.eliminate_zeros()
    with pytest.raises(ValueError, match="no path"):
        shortest_path_fast(csr2, 0, 7)


def test_helpful_error_without_scipy(monkeypatch):
    """Even with scipy installed, simulate the import-error path."""
    from cpg2unstructured import graph as g
    monkeypatch.setattr(g, "_HAS_SCIPY", False)
    with pytest.raises(ImportError, match=r"\[fast\]"):
        g.cell_adjacency_csgraph(None)
```

## Acceptance criteria

1. With scipy installed, `cell_adjacency_csgraph` returns a
   `csr_array` of shape `(n_active, n_active)`, symmetric, integer
   sums of weights match PR-10's edge total.
2. `shortest_path_fast` and `shortest_path` agree to `atol=1e-12` on
   the unit + distance weight 10×10×5 test grid.
3. Without scipy, calling either fast function raises `ImportError`
   whose message names the `[fast]` extra.
4. `tests/test_graph_fast.py` is skipped (not failed) when scipy
   absent.
5. No mandatory scipy import in `cpg2unstructured/__init__.py`.
6. `shortest_path_fast(csr, src, dst=None)` returns
   `(distances, predecessors)` (full-source mode).
7. `shortest_path_fast(csr, src, int)` returns `(dist, path)` matching
   PR-10's signature.

## Out of scope

- A* with heuristics.
- Bidirectional Dijkstra.
- Sparse matrix algebra exposure (CSR → user code).
- Choice of scipy version other than `>= 1.10` (already pinned).
- Bellman-Ford / negative weights (none in this domain).
