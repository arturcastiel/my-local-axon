## Problem

carries target/metric/budget, no mechanical state machine enforces converged/exhausted/
plateau, no receipts trail per iteration.

## Change
`tools/loop_contract.py` (NEW — Goal C's core engine; designer PR-011 builds on it):
- `define --id --target "<text>" --metric <name> --threshold X --direction min|max
  --budget N [--plateau K]` → workspace/loops/<id>.contract.json + a registered goal
  (goal set, level=workflow — reuses the existing goal store).
- `iterate --id --value X [--note]` → appends iteration; mechanical transitions:
  threshold met → CONVERGED · iterations ≥ budget → EXHAUSTED (+ report) · no
  improvement in K → advice REPLAN (allowed within budget — owner choice 2026-06-11);
  every iteration writes a loop-receipt (actor=loop-contract).
- `status/report --id` → state + best iterate + history + why-stuck.
- Budget = hard wall: `rebudget` requires literal `--human-confirm yes` + is audited;
  iterate NEVER raises budget (owner: never self-raised).
Tests: converge/exhaust/plateau paths · budget immutability · receipts written ·
report shape.
